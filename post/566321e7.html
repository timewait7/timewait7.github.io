IDEA常用快捷键

Posted on 2024-11-20 In 工具 Word count in article: 599 Reading time ≈ 2 mins.

通用操作

操作 Windows Mac
重命名 Shift+F6 ⇧+Fn+F6
跳转到某行 Ctrl+G ⌘+L
新建 ⌘+N
全局搜索/替换 Ctrl+Shift+F/R ⌃+⇧+⌘+F/R
格式化 Ctrl+Alt+L ⌘+⌥+L
import优化 Ctrl+Alt+O ⌃+⌥+O
复制行 Ctrl+D ⌘+D

光标移动

操作 Windows Mac
移动光标到代码块开始位置 Ctrl+[ ⌘+⌥+[
移动光标到代码块结束位置 Ctrl+] ⌘+⌥+]
移动光标到当前行的开始位置 Home ⌘+←
移动光标到当前行的结束位置 End ⌘+→
移动光标到上一个单词 Ctrl+← ⌥+←
移动光标到下一个单词 Ctrl+→ ⌥+→

代码块操作

操作 Windows Mac
块注释 Ctrl+Shift+/ ⌘+⇧+/
展开/折叠代码块 Ctrl+加号/减号 ⌘+加号/减号
展开/折叠全部代码块 Ctrl+Shift+加号/减号 ⌘+⇧+加号/减号

代码编辑

操作 Windows Mac
字面量切换大小写 Ctrl+Shift+U ⌘+⇧+U
删除光标后面的单词 Ctrl+Delete ⌥+Fn+⌫
删除光标前面的单词 Ctrl+BackSpace ⌥+⌦
包围代码 Ctrl+Alt+T ⌘+⌥+T
生成代码 Alt+Insert ⌃+Enter
覆盖方法 Ctrl+O ⌘+O
实现方法 Ctrl+I ⌘+I
更改签名 Ctrl+F6 ⌘+Fn+F6
提取方法 Ctrl+Alt+M ⌘+⌥+M
提取变量 Ctrl+Alt+V ⌘+⌥+V
提取字段 Ctrl+Alt+F ⌘+⌥+F
提取参数 Ctrl+Alt+P ⌘+⌥+P
向上插入一行 Ctrl+Alt+Enter ⌘+Enter
向下插入一行 Shift+Enter ⇧+Enter

类和方法

操作 Windows Mac
弹窗显示当前类结构 Ctrl+F12 ⌘+Fn+F12
查看继承树结构 Ctrl+H ⌃+H
类的UML关系图 Ctrl+Alt+U ⌥+⌘+U
弹窗显示或跳转到父类（接口） Ctrl+U ⌘+U
弹窗显示或跳转到子类 Ctrl+Alt+B ⌥+⌘+B
弹窗显示或跳转到方法的子类实现 Ctrl+Alt+B ⌥+⌘+B
跳转到选择目标的定义处或使用处 Ctrl+B ⌘+B
查看方法调用链 Ctrl+Alt+H ⌃+⌘+H
形参提示 Ctrl+P ⌘+P
查看文档 Ctrl+Q ⌃+J

代码调试

操作 Windows Mac
Run启动 Shift+F10 ⌃+R
Debug启动 Shift+F9 ⌃+D
停止 Ctrl+F2 ⌘+Fn+F2
下一步（Step Over） F8 Fn+F8
下一步（Step In） F7 Fn+F7
选择进入哪个方法 Shift+F7 ⇧+Fn+F7
跳出（Step Out） Shift+F8 ⇧+Fn+F8
恢复运行 F9 ⌥+⌘+R
运行到光标处 Alt+F9 ⌥+Fn+F9
加上/取消断点 Ctrl+F8 ⌘+Fn+F8
查看所有断点 Ctrl+Shift+F8 ⌘+⇧+Fn+F8
计算表达式 Alt+F8 ⌥+Fn+F8

拉取代码 Ctrl+T ⌘+T
弹出git操作 Alt+` ⌃+V
