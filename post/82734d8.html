Posted on 2024-08-27 Edited on 2025-01-01 In Linux Word count in article: 1.1k Reading time ≈ 4 mins.

文件系统

目录结构

Linux文件系统目录结构通常如下：

目录说明：

  • /bin：存放基本的用户命令文件，如ls、cp等
  • /boot：存放启动系统所需文件
  • /dev：设备文件
  • /etc：存放系统和应用程序的配置文件
  • /home：存放用户的主目录
  • /lib：存放系统程序和内核所需的共享库文件
  • /media：系统自动识别移动设备并挂载这个目录
  • /mnt：将外部存储挂载到当前目录
  • /opt：存放可选的第三方软件包
  • /proc：虚拟文件系统，提供关于内核和进程的信息
  • /root：超级用户（root）的主目录
  • /sbin：存放系统管理命令和工具
  • /srv：存放系统提供的服务的数据，如Web服务器的数据
  • /tmp：存放临时文件
  • /usr：存放应用程序和库，包括大多数系统工具和应用程序
  • /var：存放动态数据，如日志、缓存、邮件等
  • /usr/local：存放通过源码安装的软件

文件基本属性

  • [d] 目录
  • [-] 文件
  • [l] 链接
  • [b] 存储设备
  • [c] 串行端口设备，如鼠标、键盘

常用命令

文件和目录操作

cd

切换目录

命令 说明
cd - 返回上次所在的目录

pwd

mkdir

命令 说明
mkdir -P dir1/dir2/dir3 递归创建目录

rmdir

ls

命令 说明
ls -a 显示所有文件（包括隐藏文件）
ls -l 显示详细信息
ls -F 在文件名后附加一个指示符来标识文件类型
ls -1 强制每个文件单独占一行显示
ls -R 递归列出所有文件和目录

cp

命令 说明
cp -r dir1 dir2 递归复制（保留基本结构）
cp -a dir1 dir2 归档复制（保留元数据）

mv

rm

touch

touch test_one：创建空文件

touch test_one：改变文件的修改时间

touch -a test_one：改变文件的访问时间

ls -l test_one：时间列展示的是文件的修改时间

ls -l --time=atime test_one：时间列展示的是文件的访问时间

ln

命令 说明
ln file hl_file 创建硬链接
ln -s file sl_file 创建软链接

文件搜索

find

命令 说明
find / -name filename 从 / 开始进入根文件系统搜索文件和目录
find / -type f 搜索文件
find / -name \*.bin 搜索以 .bin 结尾的文件

locate

命令 说明
locate \*.bin 搜索以 .bin 结尾的文件

whereis

命令 说明
whereis halt 显示一个二进制文件、源码或man的位置

which

命令 说明
which halt 显示一个二进制文件或可执行文件的完整路径

-n：显示的时候添加行号

-b：只给非空行添加行号

cat /dev/null > file：清空文件

less

head

tail

系统监控和管理

top

实时显示系统进程和资源使用情况

ps

查看系统运行中的进程信息

常用参数

  • -a：显示所有进程，除会话首进程和无终端进程
  • -u：显示进程的详细信息
  • -x：显示进程运行时的参数
  • -e/-A：显示所有进程
  • -f：以完整格式显示，包括父进程ID

df

显示文件系统的磁盘空间使用情况

du

显示文件或目录的磁盘使用情况

uptime

显示系统的运行时间和负载
