文件系统

目录结构

Linux文件系统目录结构通常如下：

目录说明：

  • /bin：存放基本的用户命令文件，如ls、cp等
  • /boot：存放启动系统所需文件
  • /dev：设备文件
  • /etc：存放系统和应用程序的配置文件
  • /home：存放用户的主目录
  • /lib：存放系统程序和内核所需的共享库文件
  • /media：系统自动识别移动设备并挂载这个目录
  • /mnt：将外部存储挂载到当前目录
  • /opt：存放可选的第三方软件包
  • /proc：虚拟文件系统，提供关于内核和进程的信息
  • /root：超级用户（root）的主目录
  • /sbin：存放系统管理命令和工具
  • /srv：存放系统提供的服务的数据，如Web服务器的数据
  • /tmp：存放临时文件
  • /usr：存放应用程序和库，包括大多数系统工具和应用程序
  • /var：存放动态数据，如日志、缓存、邮件等
  • /usr/local：存放通过源码安装的软件

文件基本属性

文件类型

  • [d] 目录
  • [-] 文件
  • [l] 链接
  • [b] 存储设备
  • [c] 串行端口设备，如鼠标、键盘

在 Linux 中，每个文件和目录都有三类权限：

  • **读权限 (r)**：允许读取文件内容（对于目录，允许列出目录中的文件）。
  • **写权限 (w)**：允许修改文件内容（对于目录，允许在目录中创建、删除文件）。
  • **执行权限 (x)**：允许执行文件（对于目录，允许进入目录或访问其中的文件）。

这些权限是分配给以下三种用户类型的：

  • **文件所有者 (Owner)**：文件的创建者。
  • **文件所属组 (Group)**：文件所属的用户组。
  • **其他用户 (Others)**：除文件所有者和所属组之外的所有其他用户。

以截图中的文件 dockerfile 为例：

  • 文件类型为 d，即目录
  • 文件所有者权限为 rwx，即可读可写可访问
  • 文件所属组权限为 r-x，即可读不可写可访问
  • 其他用户权限为 r-x，也是可读不可写可访问

常用命令

文件和目录操作

cd

切换目录

命令 说明
cd - 返回上次所在的目录

pwd

mkdir

命令 说明
mkdir -P dir1/dir2/dir3 递归创建目录

rmdir

ls

命令 说明
ls -a 显示所有文件（包括隐藏文件）
ls -l 显示详细信息
ls -F 在文件名后附加一个指示符来标识文件类型
ls -1 强制每个文件单独占一行显示
ls -R 递归列出所有文件和目录
ls -t 按修改时间倒序显示
ls -tr 按修改时间正序显示

cp

命令 说明
cp -r dir1 dir2 递归复制（保留基本结构）
cp -a dir1 dir2 归档复制（保留元数据）
cp -u file1 file2 有更新才复制

mv

命令 说明
mv -i 移动前提示用户确认

rm

命令 说明
rm -i 删除前提示用户确认
rm -r 递归删除
rm -f 不提示，强制删除

touch

命令 说明
touch file 创建空文件
touch existed_file 更新文件的修改时间
touch -a existed_file 更新文件的访问时间

ln

命令 说明
ln file hl_file 创建硬链接
ln -s file sl_file 创建软链接

文件搜索

find

命令 说明
find /path -name filename 搜索当前目录及其子目录
find /path -maxdepth 1 -name filename 仅搜索当前目录
find /path -mindepth 2 -name filename 从指定深度开始搜索
find /path -type f 搜索文件
find /path -iname filename 忽略大小写

locate

命令 说明
locate -i filename 忽略大小写
locate -r pattern 正则匹配查找

which

which 用于显示某个命令或程序的绝对路径。它会查找系统的 $PATH 环境变量中列出的目录，并返回第一个匹配的命令或程序的路径。

命令 说明
which python 显示一个二进制文件或可执行文件的完整路径

whereis

whereis 是一个类似于 which 的命令，用于查找命令、二进制文件、源代码和手册页的所在位置。与 which 只查找 $PATH 环境变量中的可执行文件不同，whereis 会搜索系统中更多的路径，包括标准二进制文件路径、源代码目录和手册页目录。

命令 说明
whereis python 输出会包括多个路径，显示命令、源代码和手册页的路径
whereis -b python 查找 python 的二进制文件
whereis -m python 查找 python 的手册页

文件内容查看和编辑

wc

统计文本，输出3个值：文本行数、文本词数、文本字节数

cat

命令 说明
cat -n file 显示时添加行号
cat -b file 空行不添加行号
cat /dev/null > file 清空文件

more

less

head

命令 说明
head -n 5 file / head -5 file 显示文件前5行

tail

命令 说明
tail -n 5 file / tail -f file 显示文件最后5行
tail -f file 实时监控文件更新

sort

grep

命令 说明
grep "keyword" file 查找包含 keyword 的行
grep -i "keyword" file 忽略大小写
grep -n "keyword" file 查找包含 keyword 的行并显示行号
grep "^keyword" file 查找以 keyword 开头的行
grep "keyword$" file 查找以 keyword 结尾的行
grep "[0-9]" file 查找包含数字的行
grep -E 支持更强大的扩展正则表达式
grep -r "keyword" . 在当前目录及其子目录查找
grep -A 10 "keyword" file 显示匹配行后面 n 行
grep -B 10 "keyword" file 显示匹配行前面 n 行
grep -C 10 "keyword" file 显示匹配行前后各 n 行

PS：想高效地使用 grep，需要熟练掌握正则表达式。

系统监控和管理

ps

命令 说明
ps -aux -a：显示所有用户的进程
-u：显示进程的详细信息，特别是进程所属的用户（USER），CPU 和内存占用情况（%CPU、%MEM）
-x：包括后台进程
ps -ef -e：显示所有进程，等同于 -A
-f：以树状结构显示进程，显示进程的父子关系，并且包括更多的详细信息，如父进程 ID（PPID）、终端（TTY）等

小结：

  • ps -aux 显示详细的进程信息（包括资源占用），没有父子进程关系。

  • ps -ef 显示进程树，重点展示进程间的层次结构

top

top 用于实时显示系统的资源使用情况，包括 CPU、内存、进程等信息。

df

df 用于显示文件系统磁盘空间使用情况，包括已用空间、可用空间和文件系统总容量等信息。

命令 说明
df -h 显示可读性更好的输出

du

du 用于显示文件和目录的磁盘使用情况，可以帮助查看文件或目录占用的磁盘空间，以便有效地管理磁盘空间。

命令 说明
du -h /path 查看 /path 目录的磁盘使用情况
du -sh /path 显示目录的总磁盘使用情况

uptime

显示系统的运行时间和负载。

网络操作

ping

ping 是一个网络诊断工具，用于测试主机之间的网络连通性，以及通信的延迟情况。

命令 说明
ping -c 3 baidu.com 发送指定数量的请求

ifconfig/ip

ifconfig（Interface Configuration）用于配置和显示网络接口信息。在现代Linux系统中，ifconfig 逐渐被 ip 命令所取代，但仍然广泛使用。

netstat/ss

netstat（network statistics）用于显示网络连接、路由表、接口统计、伪装连接和多播成员的信息。它提供了一个概述当前系统网络状态的方式，帮助用户了解哪些端口正在被使用、连接的状态以及网络接口的统计数据。

curl

下载文件或测试网络请求

wget

下载文件或测试网络请求

压缩和解压缩

tar

常用选项：

  • -c：创建一个新的归档文件。

  • -x：解压归档文件。

  • -t：查看归档文件的内容。

  • -v：显示详细信息

  • -f：指定归档文件名。

  • -z：使用 gzip 压缩。

  • -C：解压到指定目录。

命令 说明
tar -cvf archive.tar file1 file2 dir1 创建未压缩归档文件
tar -czvf archive.tar file1 file2 dir1 创建压缩归档文件
tar -xvf archive.tar 解压未压缩归档文件
tar -xzvf arvhive.tar 解压压缩归档文件

zip/unzip

用于创建和解压缩 zip 文件

gzip/gunzip

用于压缩和解压缩 gzip 文件

其他命令

history

命令 说明
history -5 显示最近5条命令
!10057 执行编号为10057的命令
!man 调出最近一条 man 命令
!! 调出最近一条命令

history | grep man：查找包含 man 的历史命令

PS：命令历史通常保存在 ~/.bash_history 文件中

(un)alias

命令 说明
alias 查看所有别名
unalias ll 删除别名
alias ll='ls -l' 设置别名

在命令行通过 alias 设置的别名只在当前 shell 有效，要让别名永久生效，需要在配置文件中设置，用户的 shell 配置文件通常是~/.bashrc 或 ~/.bash_profile。

echo

命令 说明
echo -n > file 清空文件
echo -n "prompt: "; command 在同一行显示提示和命令输出
echo $? 查看shell执行的最后一条命令的退出状态码

set

命令 说明
set 显示当前的完整环境变量列表
