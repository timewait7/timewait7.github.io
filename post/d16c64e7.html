vim基础

Posted on 2024-12-20 In Linux Word count in article: 856 Reading time ≈ 3 mins.

基本命令

常用

命令 说明
i 在光标前插入
a 在光标后插入
o 在当前行后插入一行
O 在当前行前插入一行
esc 返回Normal模式（退出编辑）
:q 退出
:q! 强制退出
:wq 保存并退出
hjkl ←↓↑→
u 撤销上一步操作
ctrl+r 重放上一步操作
:set number 设置行号
:set nonumber 取消行号
G 跳转到最后一行开头
gg 跳转到第一行开头
:n + enter 跳转到指定行
n + G 跳转到指定行

光标移动

命令 说明
0 移动光标到行头
$ 移动光标到行尾
^ 移动光标到第一个非空字符
g_ 移动光标到最后一个非空字符
w 移动光标到下一个单词开头
b 移动光标到单词开头
B 移动光标到单词开头，包含标点符号
e 移动光标到单词结尾
E 移动光标到单词结尾，包含标点符号

选择、剪切、拷贝

光标命令 删除命令 选择命令 剪切命令 拷贝命令
dd yy
0 c0 v0 d0 y0
$ c$ v$ d$ y$
w cw vw dw yw
b cb vb de ye
B cB vB dE yE
e ce ve db yb
E cE vE dB yB
G cG vG dG yG
gg cgg vgg dgg ygg

删除字符

命令 说明
x 删除当前字符
X 删除前一个字符

复制

命令 组合命令 说明
p x/X + p 小写p，将剪切版内容复制到当前光标的后面
d./y. + p 小写p，将剪切版内容复制到当前光标的下一行
P x/X + P 大写P，将剪切版内容复制到当前光标的前面
d./y. + P 大写P，将剪切版内容复制到当前光标的上一行

搜索

命令 说明
/keyword + enter 向下搜索
?keyword + enter 向上搜索
n 下一个匹配项
N 上一个匹配项

进阶命令

重复

命令 说明
. 重复上一次命令
n<command> 重复某个命令n次

光标移动

命令 说明
) 移动光标到下一个句子开头
( 移动光标到上一个句子开头
} 移动光标到下一段落开头
{ 移动光标到上一段落开头
* 移动光标到下一个匹配单词
# 移动光标到上一个匹配单词
f<char> 移动光标到下一个<char>
t<char> 移动光标到下一个<char>的前一个字符

搜索

命令 说明
:set hlsearch 开启高亮显示
:nohlsearch 关闭高亮显示
/\<keyword\> 全词匹配搜索
:set noignorecae 大小写敏感（默认）
:set ignorecase 忽略大小写
/\d 搜索包含数字的行
/^keyword 搜索以keyword开头的行

搜索并替换

命令 说明
:%s/foo/bar/g 全文替换
:%s/foo/bar/gc 替换前确认
:%s/foo/bar/gi 忽略大小写
:10,20s/foo/bar/g 指定范围替换
:.,$s/foo/bar/g 从当前行到文件末尾替换
:%s/foo//g 删除匹配文本

大小写切换

命令 说明
gu+l(→) 当前字符变小写
gu+enter 当前行变小写
gU+l(→) 当前字符变大写
gU+enter 当前行变大写

实用命令

命令 说明
:% 选中全文
:%d 删除全文
:%y 拷贝全文到寄存器
:%y+ 拷贝全文到系统剪切板
ggVGy 拷贝全文到寄存器
dt" 删除所有的内容，直到遇到双引号
y2/foo 拷贝两个 foo 之间的内容
